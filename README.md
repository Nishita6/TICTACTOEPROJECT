<h1 align="center">❌ Tic-Tac-Toe Game ⭕</h1>

<p align="center">
  <strong>A classic, interactive two-player Tic-Tac-Toe game built entirely in Python.</strong>
</p>

<p align="center">
  Features clean command-line interface execution, real-time board updates, and automatic win/tie detection logic.
</p>

---

## Tech Stack & Features

* **Language:** Python 3.x
* **Core Logic:** Matrix/Grid-based state tracking, conditional win-checking algorithms, and turn-based loop control.
* **Interface:** Clean, dynamic Command Line Interface (CLI).
