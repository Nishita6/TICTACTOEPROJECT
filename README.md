<h2 align="center">❌ Tic-Tac-Toe Game ⭕</h2>

<p align="center">
  <strong>A classic, interactive two-player Tic-Tac-Toe game built entirely in Python.</strong>
</p>

<p align="left">
  Features clean command-line interface execution, real-time board updates, and automatic win/tie detection logic.
</p>

---

## Tech Stack & Features

* **Language:** Python 3.x
* **Core Logic:** Matrix/Grid-based state tracking, conditional win-checking algorithms, and turn-based loop control.
* **Interface:** Clean, dynamic Command Line Interface (CLI).

---

## Repository Structure

```text
├── tictactoe.py       # Main Python script containing the game logic and loop
└── README.md          # Project documentation

```
## How to Play (Local Setup)

* **Clone the Repository**
```text
git clone https://github.com/Nishita6/TICTACTOEPROJECT.git
cd TICTACTOEPROJECT
```

* **Run the game**
```text
python tictactoe.py
```
